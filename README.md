# E-Shop Android Admin

Native Android admin application for the e-shop project.

## Android Admin Development Docs

Start here before implementing any Android admin feature:

- [Android Admin Development Guide](docs/android-admin-development.md)
- [Android Admin Feature Checklist](docs/android-admin-feature-checklist.md)

The app should be built with Java and XML in the classroom style shown in `ref-android`. Use `ref/e-shop/admin/src` for React admin behavior parity and `ref/e-shop/backend/e-shop/docs` for backend API contracts.
